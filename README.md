# learn-docker-nginx-uwsgi-flask
> Getting to know Docker containerization of a Flask web app served with Nginx and uWSGI.

## Requirements

- docker

## Usage

### Start Server

For server in dev mode:

```sh
source dev.sh
```

For production server:

```sh
source prod.sh
```

### Navigate to localhost

Dev server:

```sh
[http://localhost:4000]
```

Prod server:

```sh
[http://localhost]
```
